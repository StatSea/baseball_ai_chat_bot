[팀원 전달용 가이드]

안녕하세요! 챗봇 API 연결 작업을 위해 전달드리는 파일입니다.

1. 작업 대상 파일: js/chat.js
- 이 파일의 `callWantedLaaSAPI` 함수(약 63번 줄)를 수정해주시면 됩니다.
- 제가 해당 위치에 `[팀원 전달 사항]`이라는 주석과 함께 설정 변수(`apiUrl`, `headers`)를 만들어 두었습니다.
- 원티드 LaaS에서 배포하신 API 주소와 Key를 그곳에 넣어주세요.

2. 실행 방법
- 웹 서버가 따로 필요 없다면 `index.html` 파일을 크롬 브라우저로 열어서 바로 테스트해보실 수 있습니다.
- 만약 로컬 서버가 필요하다면 `project` 폴더 없이, 이 폴더(`ai_baseball`) 루트에서 서버를 띄우시면 됩니다. (예: `python -m http.server` 등)

3. 참고
- 현재 CORS 문제 방지를 위한 로컬 프록시 코드는 주석 처리하고, 직접 API를 호출하는 구조로 변경해 두었습니다.
- 테스트 시 브라우저 콘솔(F12)을 켜두시면 API 응답 데이터를 확인하실 수 있습니다.

감사합니다!
